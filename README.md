# TrackOps

Updated React/Vite website for Olympus Pines leaders.

## Updates Included

- Home page has a weekly announcement box with rounded corners.
- Announcement tab removed.
- Add Task button added to the top bar.
- Home page now includes a current-week calendar.
- Home page allows task creation.
- Completed tasks record who closed the ticket and when.
- Default weekly tasks are color-coded red.
- Manually created tasks are color-coded blue-gray to match the layout.
- Tasks tab now shows task history and tickets closed by each person.
- Leaders tab renamed to Team.
- Team page displays the site team and task completion history.
- Detergent Shipment quick task included.

## Run Locally

```bash
npm install
npm run dev
```
